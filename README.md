# 💳 Credit Card Analysis & Prediction App

An interactive web application built with Python and Streamlit designed to analyze credit card data, visualize key financial insights, and provide data-driven predictions.

## 🚀 Live Demo
You can access and interact with the live application here:  
👉 **[Launch Streamlit App](https://credit-card-tugwfdaphvedxuxcljvlq4.streamlit.app/)**

---

## ✨ Features
* **Interactive Dashboard:** Dynamic data filtering and real-time metric updates.
* **Data Visualization:** Clean, interactive charts and graphs to understand user trends, spending habits, or risk factors.
* **Predictive Insights:** Built-in data modeling to evaluate credit card trends and metrics.
* **User-Friendly Interface:** Simple sidebar inputs allowing users to test different data scenarios instantly.

---

## 🛠️ Built With
* **Python** - Core application logic.
* **Streamlit** - Web framework for rapid UI deployment.
* **Pandas & NumPy** - Data manipulation and engineering.
